# react-native-matrix-rust-sdk

React Native bindings for the Matrix Rust SDK

## Installation

```sh
npm install react-native-matrix-rust-sdk
```

## Usage


```js
import { multiply } from 'react-native-matrix-rust-sdk';

// ...

const result = multiply(3, 7);
```
